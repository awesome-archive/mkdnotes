# 生成器命令 - --help 列出帮助信息 - --version 当前生成器版本号 - --init [-o] filename 在当前目录下创建一个笔记模板文件，添加"-o"选项，你的配置文件将会被读取 - --commit 根据当前的笔记目录结构重新生成数据文件data.js # 笔记主题 - amelia - bootstrap - bootstrap-responsive - cerulean - cyborg - journal - readable - simplex - slate - spacelab - spruce - superhero - united # 笔记目录主题 - theme-light - theme-dark - theme-white - theme-black
